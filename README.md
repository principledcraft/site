# principledcraft.dev

Source for [principledcraft.dev](https://principledcraft.dev), a personal site for structuring and collecting my thoughts on software engineering principles, covering topics like code quality, testing, and recommended reading.

Built with [Hugo](https://gohugo.io/) and the [Hextra](https://github.com/imfing/hextra) theme.

## Contributing

Feel free to open a PR or submit an issue if you want to contribute or leave a comment. I will review it, but since this is a personal project I can give no guarantee if or how fast it will be picked up.
