<p class="book-meta">Max Kanat-Alexander &nbsp;·&nbsp; O'Reilly Media &nbsp;·&nbsp; April 2012 &nbsp;·&nbsp; <a href="https://www.codesimplicity.com/book/">Available here (free PDF)</a></p>

A short, focused book that distils software design into a set of logical principles. It argues that the purpose of software is to help people, and that good design is measured by how easy the code is to change over time. Complexity causes most software problems, so every design decision should weigh immediate benefit against long-term cost. Rather than prescribing specific techniques, the book focuses on the underlying laws of software.
